# Restaurant Ordering Frontend

Frontend for restaurant ordering system built with React and Vite.

## Features

- User authentication
- Menu browsing
- Cart management
- Order placement
- Admin and staff dashboards

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Tech Stack

- React 18
- React Router
- Axios
- Vite
